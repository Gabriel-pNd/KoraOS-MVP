# Phases

Esta pasta organiza os artefatos por fase do projeto.

Estrutura inicial:
- `00-PRD`
- `01-UX`
- `02-Architecture`
- `03-Product-Consolidation`
- `04-PO-Sharding`
- `05-Implementation`

Convencao:
- Cada fase deve manter seus entregaveis no proprio diretorio.
- Handoffs entre fases ficam em `Docs/Handoffs/`.
- `Docs/PRD/` permanece como base canonica da Fase 00 (PRD).
