# Handoffs Versionados

Esta pasta guarda handoffs versionados e imutaveis.

Regra de uso:
- Nunca sobrescrever um handoff existente.
- Cada novo handoff deve ser um novo arquivo.
- Nome recomendado: `HANDOFF_YYYY-MM-DD_PHASE-XX-NOME_vN.md`
- Atualizar `LATEST.md` para apontar para a versao mais recente.

Primeiro handoff versionado criado a partir do canonic:
- `HANDOFF_2026-02-09_PHASE-00-PRD-CANONICAL_v1.md`
